## Общая информация

Главная страница платформы предоставляет доступ ко всем функциям и событиям.

### Элементы интерфейса

Основные элементы главной страницы:

1.  **Панель навигации**:
    -   **Доступные мероприятия**: Онлайн и оффлайн события.
    -   **Культурные мероприятия**: Сопровождающие события (музеи, парки) и достопримечательности.
2.  **Поиск и фильтры**: Инструменты для выбора интересующих событий.
3.  **Лента событий**: Карточки предстоящих мероприятий.

### Поиск и фильтрация

Для удобного поиска мероприятий предусмотрены следующие фильтры:

| Фильтр | Описание |
| :---: | :--- |
| **Название** | Поиск по наименованию мероприятия |
| **Дата** | Фильтрация по дате начала или окончания |
| **Время** | Выбор временного интервала проведения |
| **Спикеры** | Поиск по имени одного или нескольких спикеров |
| **Теги** | Фильтрация по тематическим тегам |

Также доступна **Сортировка** событий и **Глобальный поиск** по сайту, который ищет совпадения по всему тексту карточек.

### Карточка мероприятия

Карточка события содержит краткую информацию:
-   Название, дата, время и место проведения.
-   Спикеры и теги.
-   Кнопки действий: Регистрация, Избранное, Отзыв.

Для просмотра подробной информации нажмите на название мероприятия.

## Регистрация на мероприятие

1.  Перейдите в карточку интересующего мероприятия.
2.  Нажмите кнопку **Регистрация**.

В подробной карточке доступны:
-   Полное описание и список спикеров.
-   **Ссылка на подключение** (для онлайн) или **QR-код** (для оффлайн).
-   Галерея фотографий и документы для скачивания.

!!! tip "QR-код"
    При нажатии на QR-код он разворачивается на весь экран.

Зарегистрированные мероприятия отображаются в разделе **"Зарегистрированные мероприятия"** и в личном календаре.

### Отмена регистрации
Вы можете отменить регистрацию в любой момент, нажав кнопку **"Отменить регистрацию"** в карточке события.

## Избранное

Вы можете добавлять интересные события в "Избранное", нажав на иконку сердечка ❤.
-   Событие попадает в раздел **"Избранные"**.
-   Событие также отображается в **Календаре**.

## Личный кабинет

Доступ к личному кабинету осуществляется через верхнюю панель навигации.

### Возможности кабинета

-   **Загрузка фото**: Нажмите "Загрузить фото" для установки аватара.
-   **Смена пароля**: Нажмите "Сменить пароль". Новый пароль будет отправлен вам ботом в Telegram.
-   **Запрос прав администратора**: Введите обоснование в поле "Запрос админских прав" и отправьте заявку.

## Связь с администраторами

Если у вас возникли вопросы по мероприятию:

1.  Откройте карточку мероприятия.
2.  Нажмите кнопку связи с администратором.
3.  Ваше сообщение будет отправлено в группу администраторов.
4.  Администраторы получат ссылку на ваш профиль и смогут ответить вам в личном чате.

## Навигация по возможностям

- Дашборд событий с фильтрами по дате, формату, департаменту.
- Рекомендательная лента с учётом интересов и истории.
- Интеграция с календарями (ICS/Google).

## Лента событий

!!! tip "Смарт-фильтры"
    Используйте быстрые пресеты: «Сегодня», «Хочу онлайн», «Нужен сертификат».

- Карточка события показывает квоты, формат, требуемые документы.
- Из карточки доступны действия: регистрация, добавление в избранное, вопросы организатору.

## Управление участием

| Возможность            | Детали                                                           |
| ---------------------- | ---------------------------------------------------------------- |
| Быстрая регистрация    | Один клик из карточки, подтверждение в боте                     |
| Верификация документов | Загрузка справок/приглашений прямо в интерфейсе                  |
| Чек-лист посещения     | Напоминания о дедлайнах, контроле присутствия, сдаче отчётов     |

## Геймификация

- **Бейджи** за посещения ключевых событий.
- **Рейтинги** факультетов и команд.
- **Персональные подборки** за выполненные чек-листы.

## Интеграции

1. **Telegram** — мгновенные уведомления и QR-пропуска.
2. **Email** — транзакционные письма и digests.
3. **BI** — экспорт CSV/JSON для аналитики.

## Безопасность

- Авторизация через корпоративный SSO или защищённую форму.
- Все запросы обслуживаются по HTTPS (`{{ SITE_DOMAIN }}`).
- Логи событий сохраняются и доступны администраторам.

При форматировании markdown файлов в docs_src/ используй стандарты Material for MkDocs:

1. Admonitions: !!! note/tip/warning/danger/quote или ??? info для раскрывающихся блоков
2. Таблицы с выравниванием: | :---: | для центрирования
3. Код с указанием языка в тройных backticks
4. Mermaid диаграммы для визуализации
5. Изображения в assets/ с относительными путями
6. Списки с правильными отступами (4 пробела для вложенных)
7. Ссылки на другие страницы: [Текст](filename.md)
8. Макросы для переменных: {{ SITE_DOMAIN }}
9. Краткие абзацы, активный залог
10. YAML frontmatter в начале файла (если есть)

Всегда используй современные возможности Material for MkDocs для красивого и читаемого форматирования.

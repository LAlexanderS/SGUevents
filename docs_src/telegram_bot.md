## Основные сценарии

| Команда                | Что делает                                               |
| ---------------------- | -------------------------------------------------------- |
| `/events`              | Выдаёт подборку ближайших событий                        |
| `/my`                  | Показывает личный календарь и статусы регистраций        |
| `/qr`                  | Генерирует персональный пропуск для чек-ина              |
| `/admin`               | Доступ к панелям организатора/админа (если есть права)   |
| `/help`                | Краткий список возможностей                              |

!!! info "Уведомления"
    - регистрация подтверждена / отклонена;  
    - изменилось время или площадка;  
    - напоминание о чек-листе подготовки;  
    - алерты проверок безопасности/лимитов.

## Подключение

1. Авторизуйтесь на [{{ SITE_DOMAIN }}]({{ SITE_DOMAIN }}).
2. В профиле откройте вкладку **Телеграм** → **Привязать**.
3. Нажмите кнопку `Связать` и отправьте код в бот.
4. Получите сообщение «Связка активна».

!!! danger "Безопасность"
    Никогда не пересылайте код третьим лицам. Администраторы не спрашивают его в личных сообщениях.

## Deep links

- `{{ TG_BOT_URL }}/?start=event_<ID>` — мгновенная регистрация на событие.
- `{{ TG_BOT_URL }}/?start=checkin_<ID>` — режим сканера для волонтёров.

## FAQ

??? question "Поддерживает ли бот inline-режим?"
    Да. Введите `@sgu_events_bot <название>` в любом чате, чтобы поделиться карточкой события.

??? question "Можно ли отключить уведомления?"
    Да, через `/settings` в боте или через профиль в вебе.
